# Unity3D Shaders

Simple shaders for Unity3D that I created for games, for a challenge or following tutorials.

For now:

Shaders:
- Parallax Occlusion
- Grass Displacement
- Basic Occlusion Outline
- Tube TV Screen
- Dissolving
- Hologram
- Ice
- Shockwave (+ C# script)
- Toon Cascade
- Toon Fire Tornado
- Water

Includes:
- Simple noise functions (to use noise without a texture)
- Fractal Brownian Motion functions.
